Vue Carbon 使用文档

version 0.4.0

安装

npm install vue-carbon --save

快速开始

import Vue from 'vue'
import VueCarbon from 'vue-carbon'
import 'vue-carbon/dist/vue-carbon.css' // 加载css文件
Vue.use(VueCarbon)
